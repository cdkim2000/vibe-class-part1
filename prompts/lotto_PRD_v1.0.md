# 🎱 로또 번호 추천 시스템
## Product Requirements Document (PRD)
**버전 1.0 | 2025년 5월**

---

## 1. 프로젝트 개요

### 1.1 배경 및 목적

본 문서는 로또 6/45 번호 추천 시스템의 기능 요구사항을 정의합니다.

사용자가 원하는 번호를 직접 지정(수동)하거나, 시스템이 제안하는 번호를 선택(반자동)할 수 있도록 하여 번호 선택의 편의성과 전략적 유연성을 제공합니다.

### 1.2 대상 사용자

- 매주 로또를 구매하는 일반 사용자
- 특정 번호를 고정/제외하며 패턴을 만들고 싶은 사용자
- 빠르게 랜덤 번호를 생성하고 싶은 사용자

## 3. 상세 기능 요구사항

### 3.1 F-01 · 제외 번호 선택

#### 기능 설명
1부터 45까지의 번호 버튼이 격자 형태로 나열되며, 사용자가 번호를 클릭하면 해당 번호가 제외 목록에 추가됩니다. 제외된 번호는 시각적으로 구분(예: 회색 처리)되어 표시됩니다.

#### 상세 요구사항
- 번호 버튼 클릭 시 제외 상태 토글 (선택 → 해제)
- 제외 번호는 최대 39개까지 선택 가능 (최소 6개 번호 보장)
- 제외된 번호는 번호 생성 시 후보에서 완전히 배제
- 제외 번호 영역에 현재 선택된 번호 목록 표시

---

### 3.2 F-02 · 추천(고정) 번호 선택

#### 기능 설명
사용자가 특정 번호를 '반드시 포함'으로 지정합니다. 해당 번호는 항상 결과에 포함되며, 나머지 자리는 랜덤으로 채워집니다.

#### 상세 요구사항
- 추천 번호는 최대 5개까지 선택 가능 (나머지 1개는 랜덤)
- 추천 번호와 제외 번호는 중복 선택 불가 (상호 배타적)
- 추천 번호로 지정 시 자동으로 제외 목록에서 제거
- 추천 번호는 결과 출력 시 강조 표시 (예: 골드 컬러)

---

### 3.3 F-03 · 번호 생성

#### 기능 설명
'번호 추천' 버튼 클릭 시, 설정된 조건(제외/추천 번호)을 반영하여 6개의 번호를 오름차순으로 출력합니다.

#### 생성 알고리즘
1. 후보 풀 = 1~45 에서 제외 번호를 제거한 집합
2. 추천 번호를 결과에 우선 포함
3. 나머지 자리는 후보 풀에서 무작위 추출 (중복 없음)
4. 최종 결과 6개 번호를 오름차순 정렬하여 표시

#### 유효성 검사
- 후보 풀 < 6개인 경우 경고 메시지 출력 후 생성 중단
- 추천 번호 > 5개인 경우 생성 버튼 비활성화

## 4. 사용자 시나리오

### 4.1 수동 모드 (번호 직접 고정)

1. 사용자가 원하는 번호 (예: 7, 14, 22)를 추천 번호로 클릭
2. 절대 나오지 않았으면 하는 번호 (예: 1, 2, 3)를 제외 번호로 클릭
3. '번호 추천' 버튼 클릭 → 7, 14, 22 포함된 6개 번호 출력

### 4.2 반자동 모드 (시스템 추천 활용)

1. 아무 설정 없이 '번호 추천' 버튼 클릭
2. 출력된 결과에서 마음에 드는 번호를 추천 번호로 고정
3. 다시 '번호 추천' 버튼 클릭 → 고정된 번호 포함 새 조합 출력

## 5. UI 레이아웃 설계

퍼블리싱 환경: 순수 바닐라 JavaScript + HTML + CSS (프레임워크 사용 없음)

### 5.1 화면 구성

| 영역 | 설명 |
|------|------|
| 헤더 | 서비스 타이틀 및 현재 회차 정보 |
| 번호 그리드 | 1~45 번호 버튼 (7×7 또는 9×5 그리드) |
| 모드 레이블 | 클릭 모드 선택: 제외 번호 / 추천 번호 토글 버튼 |
| 선택 현황 | 제외 번호 목록 / 추천 번호 목록 표시 뱃지 |
| 생성 버튼 | '번호 추천' CTA 버튼 (대형) |
| 결과 영역 | 생성된 번호 볼 형태로 표시, 복사 버튼 포함 |

### 5.2 번호 볼 색상 규칙

| 번호 범위 | 색상 | 비고 |
|-----------|------|------|
| 1 – 10 | 노란색 `#FBC400` | 기본 번호 볼 색상 |
| 11 – 20 | 파란색 `#069EDB` | 기본 번호 볼 색상 |
| 21 – 30 | 빨간색 `#FF6151` | 기본 번호 볼 색상 |
| 31 – 40 | 회색 `#AAAAAA` | 기본 번호 볼 색상 |
| 41 – 45 | 초록색 `#00C851` | 기본 번호 볼 색상 |
| 제외 번호 | 연회색 + 취소선 | 비활성화 표시 |
| 추천 번호 | 골드 테두리 + 별 표시 | 강조 표시 |

## 6. 기술 요구사항

### 6.1 기술 스택

- HTML5 / CSS3 / Vanilla JavaScript (ES6+)
- 외부 라이브러리 사용 금지 (Zero Dependency)
- 단일 파일 구조 (`index.html`) 권장
- 반응형 지원: 모바일(375px) ~ 데스크톱(1280px)

### 6.2 성능 요구사항

- 번호 생성 응답 시간: 100ms 이내
- 초기 로드 시간: 1초 이내
- 별도 서버 없이 파일 더블클릭으로 실행 가능

## 7. 비기능 요구사항
